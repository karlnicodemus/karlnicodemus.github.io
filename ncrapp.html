NCR Mobile Application

This is a mobile application for iOS I created during my internship at NCR Corporation. It accomplishes the following:

  • Collects the applicant's information
  • Allows recruiters to take notes on the applicant
  • Stores information in a cloud-based database
  • Parses data into a readable format

This project allowed me to explore iOS libraries and capabilities. I designed this applciation completely back to front. Using libraries for backend to design paradigms on the front, it highlights my breadth of knowledge in the mobile field.
