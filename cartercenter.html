Carter Center: Forum on Women

This is a prototyping project done in a User Interface design class. Working in a team, we collected data from our peers, used that data to evaluate the current system and later design prototypes that addressed the problems.

We emulated the entire data-driven design process. This gave me the skill to create UI that is not only aesthetically pleasing, but useful and impactful based on data.
